ubuntu安装dropbox

Veröffentlicht am 2017-12-25
ubuntu安装dropbox

此教程基于配好了shadowsocks，可进行翻墙

###dropbox下载
在官网上下载源码

解压

tar xjf ./nautilus-dropbox-1.4.0.tar.bz2

安装

cd ./nautilus-dropbox-1.4.0; ./configure; make; make install;

在进行./configure时会出错，少了一个libnautilus-extension的依赖。

用这条命令安装上

sudo apt-get install libnautilus-extension-dev

编译安装成功后，在终端输入dropbox start -i指令，可以进行dropbox的安装。

###下载 dropbox daemon
安装时dropbox提示我们需要下载东西，但是下载的东西一直下载不了。是因为ubuntu下的代理没有全局模式，连接不上dropbox.com，可以通过以下两个链接先下载下来。

32位http://www.getdropbox.com/download?plat=lnx.x86

64位http://www.getdropbox.com/download?plat=lnx.x86_64

下载完后，解压缩后的名字应该是 .dropbox-dist, 然后放在个人目录 ~/ 下面即可。

###安装proxychains
安装完后再次启动dropbox后，发现连接dropbox还是失败了，这时我们就需要通过代理让dropbox来运行。
这里可以用proxychains来启动dropbox。

sudo apt-get install proxychains

配置ssh代理信息

sudo vi /etc/proxychains.conf

在最下方添加你的代理信息

socks5 127.0.0.1 1080

然后用sudo proxychains dropbox start -i启动dropbox就可以了。
